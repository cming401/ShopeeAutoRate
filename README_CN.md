这是一个简单而强大的 Chrome 扩展程序，帮助 Shopee 卖家节省时间，自动为已完成的订单评价。

---

## ✨ 功能特点

- 🎯 **一键启动** - 自动打开 Shopee 页面并开始评价
- ⭐ **五星好评** - 自动为所有买家评五星
- ⚡ **快速模式** - 每个订单仅需约 3.2 秒
- 📊 **进度追踪** - 实时统计和活动日志
- 🔄 **多页支持** - 自动处理所有页面

---

## 📥 安装步骤

1. 点击上方绿色的 **下载扩展程序** 按钮
2. 解压 ZIP 文件
3. 打开 Chrome，访问 `chrome://extensions/`
4. 开启右上角的 **"开发者模式"**
5. 点击 **"加载已解压的扩展程序"**
6. 选择 `chrome-extension` 文件夹
7. 完成！

---

## 🎮 使用方法

1. 点击 Chrome 工具栏中的扩展图标
2. 点击 **"🚀 打开页面并自动开始"**
3. 完成！坐下来放松 ☕

扩展程序将自动打开 Shopee、为所有订单评五星，并处理所有页面。

---

## 🔧 故障排除

- **无法工作？** 刷新 Shopee 页面后重试
- **扩展图标看不见？** 点击 Chrome 工具栏的拼图图标（🧩）并固定
- **仍有问题？** 确保已登录 Shopee 卖家账号
